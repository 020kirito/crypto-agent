题目: Babypack (超递增背包)

来源: BaseCTF 2024

题目描述：
这是一个基于超递增序列的背包密码系统。

加密代码：
```python
from Crypto.Util.number import *
import random

flag = b'BaseCTF{}'
m = bytes_to_long(flag)
bin_m = bin(m)[2:]
length = len(bin_m)

a = [1]
s = 1
for i in range(length - 1):
    temp = random.randint(2*s + 1, 4*s)
    s = s + temp
    a.append(temp)

a = a[::-1]
c = 0
for i in range(length):
    if bin_m[i] == '1':
        c = c + a[i]

print("a =", a)
print("c =", c)
```

提示：
1. 这是一个超递增背包问题
2. 超递增序列：每个元素大于之前所有元素之和
3. 解密方法：从大到小贪心选择
4. 如果 c >= a[i]，则该位为 1，否则为 0

请解密密文 c 获取 flag。
