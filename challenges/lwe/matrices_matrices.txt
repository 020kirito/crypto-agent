题目: Matrices Matrices Matrices (LWE)

来源: K!nd4SUS CTF 2025

题目描述：
这是一个基于 LWE (Learning With Errors) 问题的密码系统。我们知道以下信息：

```
A * s + e = b (mod q)
```

其中：
- A 是一个 m×n 的随机矩阵
- s 是 n×1 的秘密向量 (需要恢复)
- e 是 m×1 的小误差向量
- b 是 m×1 的观测向量
- q 是模数

提示：
1. LWE 是后量子密码学的基础
2. 误差向量 e 的数量级很小
3. 可以使用 Primal Attack 恢复秘密向量 s
4. 攻击的关键是构造格使得 e 出现在短向量中

给出一个例子：
```
A = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
b = [10, 20, 30]
q = 97
n = 3 (秘密向量维度)
m = 3 (样本数)
```

请恢复秘密向量 s。
