题目: Hidden Number Problem (HNP)

来源: 做题笔记/Lattice刷题笔记.md

题目描述：
我们有一个秘密的 256 位整数 x。我们给出了 32 组 (a_i, b_i) 数据，其中：

```
a_i * x ≡ b_i (mod q)
```

其中 q = 2^256。

但 b_i 只泄露了高 8 位，即：
```
b_i = 2^248 * b_hi + b_li
```

已知数据：
```
A = [3561678147813669042672186969104055553515262226168087322052560790885260761433, 
     17346407693442644010055116546363960164095133759884497841925887458500171929994,
     ... (共32个)]

B_hi = [185, 121, 74, 192, 66, 208, 189, 5, 248, 216, 222, 49, 199, 122, 212, 109, 
        36, 135, 9, 43, 94, 192, 67, 176, 165, 34, 241, 27, 255, 216, 71, 156]
```

提示：
1. 这是一个经典的 HNP (Hidden Number Problem)
2. 利用两组数据的差分消除 x
3. 构造格并 LLL 规约
4. 从短向量恢复 b_li，进而恢复 x

请恢复秘密整数 x。
